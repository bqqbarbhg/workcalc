# workcalc
Calculates and visualizes work time sums

Example:
```
> ./workcalc.py workhours.csv

   08:05 +00:20    ............    ............    ............
   07:56 +00:11    ............    ............    ............
   08:40 +00:55    ............    ............    ............
   08:38 +00:53    ............    ............    ............
   01:50 -05:55    ............    ............    ............

   35:09 -03:36    ............    ............    ............

Month:  35:09 -03:36

Total:  35:09 -03:36
```

Requires [colorama](https://pypi.python.org/pypi/colorama) if you want to have colors.
It can be installed with:

    pip install colorama
